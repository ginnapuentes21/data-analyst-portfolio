# Análisis de Embudo y Retención para Mercado libre

## Objetivo
Analizar la retención de usuarios para identificar patrones de abandono.

## Datos utilizados
- Retención por país
- Retención por cohorte
- Conversión por país
- Conversión general

## Herramientas
- SQL
- Google sheets

## Proceso de análisis

- Explorar el esquema y los datos base.
- Construir el embudo de conversión.
- Calcular tasas y caídas.
- Analizar retención y cohortes.
- Simular mejoras y redactar el informe ejecutivo.

## Resultados

- La mayor caída porcentual de usuarios se observa entre las etapas select_item y add_to_cart, donde la conversión cae aproximadamente del 77% al 11%. Esta pérdida es consistente a nivel general, siendo Perú el país con mayor caída relativa, mientras que Uruguay presenta la menor pérdida en esta etapa.
- La retención promedio para  D7 ≈ (86–88%),a partir de D14 la retención cae a un rango cercano al 54–56%. En D21 la retención se reduce a 23–26% y para D28, la retención se estabiliza alrededor del 2–3%, lo que sugiere que solo una fracción pequeña de usuarios mantiene actividad recurrente al final del primer mes.
Se observa que México es el país que mayor retención tiene en cada uno de los días evaluados.
