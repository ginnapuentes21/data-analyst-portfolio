# Movilidad urbana y productividad económica en LATAM

## Objetivo
Evaluar la relación entre la congestión vehicular y la productividad económica en ciudades de América Latina para identificar patrones de movilidad y su posible impacto en el desarrollo urbano.

## Datos utilizados
- Índice de tráfico
- PIB per cápita
- Población
- Tiempo promedio de desplazamiento

## Herramientas
- Python
- Pandas
- Matplotlib

## Proceso de análisis
1. Limpieza de datos
2. Análisis exploratorio (EDA)
3. Análisis de correlación
4. Visualización de resultados

## Principales hallazgos
- Algunas ciudades presentan alta congestión y menor productividad.
- La movilidad urbana puede influir en la eficiencia económica de una ciudad.
